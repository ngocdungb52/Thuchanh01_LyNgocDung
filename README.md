## Mô tả nội dung thực hành
Bài thực hành này bao gồm các nội dung:
- Mô tả thông số phần cứng/ phần mềm cơ bản của máy tính đang sử dụng
- Trình bày quy trình cài đặt và các thao tác cơ bản
- Xử lí sự cố kết nối Wi-Fi, xử lí lỗi phần mềm cơ bản
- Thực hiện bài tập Python (.py) theo yêu cầu
- Ứng dụng AI trong Học tập
- Thiết kế infographic bằng Canva
- Đánh giá đạo đức khi sử dụng AI
- Chụp màn hình các công cụ: Task Manager, Notepad++, Command Prompt, Wi-Fi

## Danh sách file trong repository
- File Word/Google Docs: Tổng hợp tất cả nhiệm vụ và ảnh minh chứng ( ảnh chụp màn hình )
- File Python (.py): Bài tập nhiệm vụ 1.2
- File PNG: Infographic nhiệm vụ 3.1

## Công cụ và nguồn AI tham khảo
- Google Bard, ChatGPT (hỗ trợ giải thích và viết nội dung)
- Canva AI (hỗ trợ thiết kế infographic)
- Laptop cá nhân
